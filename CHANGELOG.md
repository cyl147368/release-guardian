## [3.1.0] - 2026-06-18

### 新增
- **前端 UI 全面重新设计 - 量子观测站主题**
  - 全新 CSS 设计令牌系统，护眼量子靛蓝色系
  - 流体曲率圆角和空间深度阴影
  - 毛玻璃效果和量子背景光效
  - 量子初始化序列动画
  - 响应式布局适配所有设备
- **WebSocket 实时推送集成**
  - 零依赖 RFC 6455 WebSocket 实现
  - 发布创建/审批/部署事件自动广播
  - 前端实时接收通知，自动刷新视图
  - 断线自动重连机制
- **数据导出功能**
  - CSV/JSON 格式导出发布记录
  - CSV/JSON 格式导出审计日志
  - 一键下载功能
- **发布流程可视化**
  - 各状态发布数量统计卡片
  - 最近活动列表
  - 状态卡片点击跳转筛选
- **审计日志查看器**
  - 完整审计日志列表视图
  - 事件类型标签和颜色标识
  - CSV/JSON 数据导出
  - 详情模态框显示完整信息
- **性能监控面板**
  - 请求分布可视化 (GET/POST/PUT/DELETE)
  - 系统健康状态检查
  - KPI 卡片 (总请求数/运行时间/延迟/错误率)
- **用户偏好设置**
  - 主题切换 (暗色/亮色)
  - 语言选择
  - 每页显示条数配置
  - 自动刷新间隔
  - 浏览器通知开关
  - 设置本地持久化
- **移动端优化**
  - 响应式布局适配所有屏幕尺寸
  - 触摸设备优化 (最小点击区域 44px)
  - iOS 输入框防缩放 (16px 字体)
  - 横屏模式优化
  - iPhone X+ 安全区域适配

### 安全改进
- 修复路径穿越漏洞，增强静态文件服务安全性
- 更新 CSP 策略以支持前端 SPA 运行
- 指标端点 /metrics 和 /api/metrics 绕过 API key 认证
- 响应自动添加 Content-Length 头

### 新增
- **Web 控制台**: 全新企业级深空主题控制台界面
  - 仪表板：KPI 卡片、风险分布图、环境分布图、状态分布图
  - 发布列表：过滤/分页/详情模态框
  - 创建发布：完整表单带控制项检查
  - 升级告警：实时风险监控
  - Webhook 管理：订阅/删除/事件日志
  - 治理策略：审批路由规则展示
  - 开机引导动画、响应式布局、Toast 通知
- **审计日志模块** (`src/lib/audit.js`)
  - 记录所有发布状态变更、审批操作和 Webhook 事件
  - 支持多维度查询（事件类型/操作者/资源类型/时间范围）
  - 新增 `/api/audit` 和 `/api/audit/stats` 端点
- **应用指标模块** (`src/lib/metrics.js`)
  - HTTP 请求计数、延迟直方图（P50/P95/P99）、热门路径
  - 业务指标：发布创建/审批/部署/SLA 违规数
  - 新增 `/api/metrics`（JSON）和 `/metrics`（Prometheus）端点
- **指标收集中间件**: 自动记录每个请求的延迟和状态码
- Docker 镜像包含前端静态资源
- CI 增加 Docker 构建验证和综合质量门禁步骤

### 改进
- 中间件管道增加指标记录层
- 审计日志在关键业务操作时自动写入
- 静态文件缓存策略优化（HTML 不缓存，其他资源缓存 1 小时）
- 安全头中间件现在允许 `style-src` 和 `script-src` 用于前端
- 版本号统一升级至 3.0.0

### 测试

- 测试总数：218 个，34 个测试套件
